ENGLISH

This page describes the privacy policy of the Android app "Discreet Launcher" (com.vincent_falzon.discreetlauncher).
Discreet Launcher is a new home screen and allows the user to access all the applications on his/her Android smartphone.

Personal information handling
Only the list of installed applications is retrieved in order to display them in the interface.
Discreet Launcher works completely offline and do not share anything with anyone.

Permissions
The Android permission "QUERY_ALL_PACKAGES" is required to get the list of applications.
The Android permission "POST_NOTIFICATIONS" is required to display the notification for quick access to favorites.
No other permissions are requested.

Data storage
Users can define settings to customize Discreet Launcher.
They are stored only in the smartphone and are removed when Discreet Launcher is uninstalled.
An export/import feature allows to get settings in a text file for backup, restoration or transfer on another device.

FRANÇAIS

Cette page décrit la politique de confidentialité de l'application Android "Lanceur Discret" (com.vincent_falzon.discreetlauncher).
Lanceur Discret est un nouvel écran d'accueil qui permet à l'utilisateur(trice) d'accéder à toutes les applications de son smartphone Android.

Traitement de données personnelles
Seule la liste des applications installées est récupérée afin de pouvoir les afficher dans l'interface.
Lanceur Discret fonctionne complètement hors-ligne et ne partage rien avec qui que ce soit.

Permissions
La permission Android "QUERY_ALL_PACKAGES" est requise pour obtenir la liste des applications.
La permission Android "POST_NOTIFICATIONS" est requise pour afficher la notification d'accès rapide aux favoris.
Aucune autre permission n'est demandée.

Stockage des données
Les utilisateurs peuvent définir des paramètres pour personnaliser Lanceur Discret.
Ils sont stockés uniquement sur le smartphone et sont supprimés quand Lanceur Discret est désinstallé.
Une fonction d'export/import permet d'obtenir ces paramètres dans un fichier texte pour sauvegarde, restauration ou transfert sur un autre périphérique.
